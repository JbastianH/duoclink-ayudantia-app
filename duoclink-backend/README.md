# DuocLink Backend

Este repositorio contiene los microservicios para el backend de DuocLink.

## Estructura

- `ms-apuntes`: Microservicio para la gestión de apuntes.
- `ms-ayudantias`: Microservicio para la gestión de ayudantías.

## Construcción

Este proyecto utiliza Maven. Puedes construir todos los módulos desde la raíz:

```bash
mvn clean install
```

O construir cada microservicio independientemente entrando en su directorio.
